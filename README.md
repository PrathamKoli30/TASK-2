# Stopwatch Web Application

A simple, lightweight, and responsive **Stopwatch Web App** built using **HTML, CSS, and JavaScript**.  
This project is ideal for beginners to understand JavaScript timing functions and DOM manipulation.

**main file : (watch.html)**

##  Features

-  Start the stopwatch
-  Pause the running timer
-  Reset the stopwatch
-  Displays time in **Minutes : Seconds : Milliseconds**
-  Clean and minimal UI
-  Responsive design
